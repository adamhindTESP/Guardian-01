# Vision Documents (Non-Normative)

This directory contains **conceptual, forward-looking documents** describing
possible future research directions for the Guardian architecture.

## Important

Documents in this directory:
- ❌ do NOT have execution authority
- ❌ do NOT advance any safety gate
- ❌ do NOT describe implemented features
- ❌ do NOT modify Guardian Seed v1.0.1 behavior

They are **not part of the certified system**.

## Certified System

The only authoritative artifacts are documented in:
- `GATES.md`
- `runtime/`
- `schema/`
- `evaluation/`

Vision documents exist solely to record long-term ideas and research questions.

Proceed accordingly.
